Linux命令

Nov 14, 2020 1 分钟阅读时长

Linux系统是后发开发必须要掌握的，而命令是其核心知识。

本文主要记录了Linux常用的命令，加以整理以防不时之需。

目录

    • apt

apt

参考：https://man.linuxde.net/apt-get-2

apt-get update
  • 获取包的最新版本，更新数据库信息
  • HIT 包版本没有变化
  • IGN 包别忽略。可能是因为包版本新，懒得去查，也可能是获取信息错误
  • GET 包版本可以升级
apt-get upgrade
  • 升级包
apt-cache search [package name]
  • 查找包
apt-cache pkgnames [package name]
  • 只查找特定包名
apt-get update && apt-get dist-upgrade
  • 快速升级到最新版本
apt-get showpkg [detail package name]
  • 查看该包的详情
apt-get install [detail package name] <...>  <...>
  • 安装包
apt-get remove [detail package name]
apt-get purge  [detail package name]
  • 删除包
  • remove 只删除二进制文件，不删除配置文件
  • purge 全部删除
